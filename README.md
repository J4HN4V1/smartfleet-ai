# SmartFleet AI

AI-powered logistics workflow platform built using MERN stack.

## Features
- Authentication
- Dashboard
- Deliveries
- Fleet Management
- Analytics
- Dispatch Board
- Profile Management

## Tech Stack
- React
- Tailwind CSS
- Node.js
- Express.js
- MongoDB

## Run Backend
cd server

npm install

npm run server

## Run Frontend
cd client

npm install

npm run dev
